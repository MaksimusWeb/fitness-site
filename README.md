# Фитнес сайт

Сайт предназначен для небольшой демонстрации моего понимания принципов работы с веб-интерфейсом приложения.

## Краткое описание

Сайт является небольшой рекомендательной шпаргалкой в сфере фитнеса. Есть:
— подсчёт BMI
— подсчёт процента жира
— разные формулы в зависимости от пола
— быстрая и удобная форма
— рекомендации на основе результатов
— другая информация сферы фитнеса
