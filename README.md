# Фитнес сайт

Сайт предназначен для небольшой демонстрации моего понимания принципов работы с веб-интерфейсом приложения.

## Возможности

- Расчет BMI и процента жира с рекомендациями
- Интерактивный подбор упражнений по группам мышц
- Тренировочные программы: набор массы, похудение, рекомпозиция
- Адаптивный дизайн
- Автоматическое развертывание на GitHub Pages

## Стек технологий

- **Frontend**: React 19.2.0, React Router DOM 7.9.4
- **Build**: React Scripts 5.0.1
- **Testing**: Testing Library
- **Deployment**: GitHub Pages, gh-pages 6.3.0

## Установка

```bash
git clone https://github.com/MaksimusWeb/fitness-site.git
cd fitness-site
npm install
npm start
```

## Деплой

```bash
npm run build
npm run deploy
```
